Card Example

To create a card use the class .card inside a div.
.card-body: to create a body inside the card.
.card-title: to add the title.
.card-img-top|bottom to add image to a card.

    
.card-link: to a link inside the card.
.card-text: text inside the body.
.stretched-link to add a link box.
